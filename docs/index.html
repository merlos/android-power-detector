Purpose-built for spare phones

Use an older Android device as a compact automation node that reacts instantly when power is connected or removed.

Clear action setup, not buried settings

Add an action from the panel, the empty state, or the floating button. Edit any action from the card list and test it before depending on it.

Telegram setup without manual retyping

Generate a QR with the Go utility, load it from the app, and prefill the Telegram bot token and chat ID in one step.

Setup Flow

Three steps from install to working automation.

01

Install the APK

Open the latest GitHub release on the target device and install Power Detector.

02

Create an action

Use the visible Add buttons to create an SMS or Telegram action for AC power or battery mode.

03

Test it immediately

Run a manual test from the form or import a Telegram QR image to avoid typing bot setup details by hand.

What It Does

Small scope, direct behavior, and no extra backend required.

  • Large, centered home screen status: AC Power or Battery
  • Configurable SMS and Telegram actions with manual test execution
  • Background execution when charging changes
  • Tap-to-edit action cards with execution history and preview text
  • Telegram setup import through a QR image generated by the included Go tool
  • Works on Android 8.1 and newer

Included Tooling

Repository includes the release pipeline and the QR generator.

You are not just downloading an APK. The repository also includes a GitHub Actions release workflow, signing documentation, GitHub Pages deployment, and a Go utility that generates Telegram setup QR codes.
